A tool for quickly finding an SVG filter for black and white images

This is a non-comprehensive list of copy and paste options for changing the color of an SVG image.

Filter black SVG green:

Copy to clipboard filter: invert(100%) brightness(40%) sepia(100%) hue-rotate(100deg) saturate(1000%) contrast(0.8);

Filter black SVG blue:

Copy to clipboard filter: invert(100%) brightness(40%) sepia(100%) hue-rotate(150deg) saturate(1000%) contrast(0.8);

Filter black SVG pink:

Copy to clipboard filter: invert(100%) brightness(40%) sepia(100%) hue-rotate(280deg) saturate(1000%) contrast(0.8);

Filter black SVG red:

Copy to clipboard filter: invert(100%) brightness(40%) sepia(100%) hue-rotate(665deg) saturate(1000%) contrast(0.8);

Filter black SVG orange:

Copy to clipboard filter: invert(100%) brightness(40%) sepia(100%) hue-rotate(715deg) saturate(1000%) contrast(0.8);
